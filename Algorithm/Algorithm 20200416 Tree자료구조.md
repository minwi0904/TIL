# 20200416 Tree자료구조

### TREE

- 비선형 구조로 원소들 간에 1:n관계를 가지는 자료구조
- 각 원소간에 계층관계를 가지는 계층형 자료구조
- 상위 원소에서 하위 원소로 내려가면서 확장되는 Tree모양구조
